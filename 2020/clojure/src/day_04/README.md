# Day 04

Passport processing.

Expected fields:

- `byr` (Birth Year)
- `iyr` (Issue Year)
- `eyr` (Expiration Year)
- `hgt` (Height)
- `hcl` (Hair Color)
- `ecl` (Eye Color)
- `pid` (Passport ID)
- `cid` (Country ID)

## Part One

If `cid` field is missing, the passport is treated as "valid".

Launch with `clj -X day-04.part-one/run`.

## Part Two

Launch with `clj -X day-04.part-two/run`.
